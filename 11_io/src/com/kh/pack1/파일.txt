File 클래스

java.io 폴더 내부에 존재하는 java파일

파일 / 디렉토리(폴더)를 관리하는 클래스
(파일 / 디렉토리 존재 유무 관계 없음)

- 파일생성, 제거, 이름, 크기, 마지막 수정일, 존재여부 등의 기능 제공

File 클래스 메서드

boolean mkdir()         :하나의 디렉토리(폴더)생성 (make directory = 만들다 폴더를)
boolean mkdirs()        :경로상의 디렉토리(폴더)생성 (make directorys = 만들다 폴더 모두 만들기)
boolean createNewFile() : 파일 생성
boolean delete()        : 파일 / 디렉토리 삭제
String getName()        : 파일 이름 반환
String parent()         : 파일이 저장된 폴더 반환
String getPath()        : 전체 경로 반환
boolean isFile()        : 현재 File 객체가 관리하는게 파일이면 true
boolean isDirectory()   : 현재 파일 객체가 관리하는게 디렉토리면 true
boolean exists()        : 파일 / 디렉토리가 존재하면 true 아니면 false
long    length          : 파일 크기 변환
long    lastModified    : 파일 마지막 수정일(1970.01.01 09:00 부터 현재까지 지난 시간을 ms단위로 반환)
String[] List()         : 디렉토리 내 파일 목록을 String[] 배열로 반환
File[]  listFiles()     : 디렉토리 내 파일 목록을 File[] 배열로 반환

C:/Users/user1/바탕화면>
C:/Users/user1/Desktop>

System.getProperty("user.home") = 현재 사용하는 유저의 바탕화면 오기 전의 폴더 위치 설정
System.getProperty("user.home") = C:/User/user1/ 여기까지 작성을 해주는 것

만약에 바탕화면에 만들고 싶다면
File 폴더 = new File(System.getProperty("user.home")+"/Desktop/newFolders");
                     C:/User/user1/ 뒤에 Desktop이라는 바탕화면 폴더에 newFolders라는 이름의 폴더를 생성할 것

getPath()           : 현재 만들고자 하는 폴더의 전체 경로를 보여주는 것
getAbsolutePath()   : 어디서 봐도 무조건적인 폴더의 전체 경로를 보여주는 것

[개발자 상식]
OS : 운영 체제 (Operating System)

        Windows             Linux/Mac
        폴더                디렉토리

     - 경로 표기 방법(하위 폴더, 파일 표시)
     window는 \(백슬래시)    C:\User\user1
  Linux,Mac은 /(슬래쉬)      C:/User/user1

  Java는 OS에 관계없이 \, / 둘다 호환해서 사용 가능
  되도록이면 경로 표기는 /로 표기

  1) 절대 경로: 하나의 (절대적인)기준으로부터 목표까지의 경로를 모두 표기하는 방법
  - 기준
      [windows] :   C:/ ,   D:/
   [Linux, Mac] : /Users,    /

  2) 상대 경로 : 현재 위치를 기준으로 목표까지의 경로를 표기하는 방법

System.out.println("폴더 위치: "+폴더.getPath());
System.out.println("폴더 절대경로: " + 폴더.getAbsoluteFile());
